# Практические работы по дисциплине Фронтенд и бэкенд разработка
Практические задания разделены по веткам репозитория

## Практическое задание 4-5

### Фронтенд
 Для запуска перейти в папку frontend и выполнить команду `pnpm install` для установки зависимостей, а затем `pnpm dev` для запуска приложения
 
### Бэкенд
  Для запуска перейти в корневую папку и выполнить команду `npm install` для установки зависимостей, а затем `npm start` для запуска приложения
